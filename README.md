Project Overview

This project demonstrates an automated testing framework built for a web-based Registration System. The objective is to validate core functionalities such as page loading, form validation, error handling, and successful data submission using Selenium WebDriver with Java, Maven, and TestNG.
The project focuses on ensuring software quality, reliability, and regression safety through automation.


Objectives

-> Automate functional testing of a registration form

-> Validate positive and negative user scenarios

-> Capture screenshots for test evidence

-> Ensure test execution is repeatable and reliable

-> Apply industry-standard automation practices

Tech Stack

Programming Language: Java

Automation Tool: Selenium WebDriver

Test Framework: TestNG

Build Tool: Maven

Browser: Google Chrome

Driver Management: WebDriverManager

IDE: Eclipse / IntelliJ IDEA


Test Scenarios Covered

-> Landing Page Verification

-> Negative Scenario Testing

-> Positive Scenario Testing

-> Form Logic Validation

-> Screenshot Capture


Prerequisites

-> Java JDK 11 or above

-> Maven installed and configured

-> Google Chrome browser

-> Stable internet connection
